# SHIKHAR OUTDOOR

A production-ready multi-page outdoor apparel website built with React and Vite. Inspired by premium alpine brands, SHIKHAR OUTDOOR ("shikhar" = peak/summit in Hindi) is designed around a Himalayan adventure aesthetic — deep forest greens, warm amber, and clean typography using Playfair Display and DM Sans.

The site includes five fully built pages — Home, Men's, Women's, Footwear, and About — with a scroll-triggered loading screen, animated hero, parallax effects, IntersectionObserver reveal animations, animated counters, sticky navbar with blur transitions, product card hover effects, a mobile hamburger drawer, and a scroll progress bar.

To run locally, install dependencies with `npm install` and start the dev server with `npm run dev`.
